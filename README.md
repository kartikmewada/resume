# Resume Website (Template)

This is a small, responsive, and print-friendly resume website template.

How to use
1. Replace placeholder text in `index.html` with your real information.
2. Add your photo at `assets/photo.jpg` (or change the path).
3. Add your PDF resume at `assets/resume.pdf` (or change the link in `index.html`).
4. Optionally change colors in `styles.css` (CSS variables at the top).
5. Serve locally: `npx http-server` or open `index.html` in a browser.
6. To host on GitHub Pages:
   - Create a repository and push these files.
   - Enable Pages in repo settings (choose `main` branch / `/ (root)`).
   - Your site will be available at `https://<username>.github.io/<repo>/`.

Features
- Responsive two-column layout (sidebar + content).
- Print-optimized output (clean, black & white).
- Theme toggle (light/dark) persisted in localStorage.
- Download resume button + Print button.
